						<h4>
							How do you mount PO.DAAC Drive to your local
							computer?
						</h4>
						<div>
    <p>PO.DAAC's WebDAV interface allows you to connect to PO.DAAC as if it were a local drive on your computer.</p>
    <p>In order to connect with WebDAV, you'll need to use your URS username and the WebDAV password that's been assigned to you.</p>
    <hr>
    <h5>Connecting via OS X</h5>
    <p>PO.DAAC Drive supports Mac OS X 10.9 and higher.</p>
    <p>From the Finder, click Go/Connect to Server, or press &#x2318;-K.</p>
    <p>In the window that appears, enter the following under <strong>Server Address</strong>:
    <code>https://podaac-tools.jpl.nasa.gov/drive/files</code></p>
    <p>Click <strong>Connect</strong>.</p>
    <p>When prompted to log in, use your URS username <strong></strong> and WebDAV password <strong></strong>.</p>
    <p>You can now browse PODAAC files with the Finder and copy them to your computer just as you would with a local hard drive. It's best to copy a file to your computer before opening it.</p>
    <p>To disconnect, drag the drive into the Trash or right-click it and select <strong>Eject</strong>.</p>
    <hr>
    <h5>Connecting via Windows</h5>
    <p><strong>Windows 10:</strong> Click <strong>Start</strong>/<strong>File Explorer</strong>. In the left-hand menu, right-click <strong>My PC</strong> and select <strong>Map Network Drive.</strong></p>
    <p>Choose the desired drive letter you'd like to see PO.DAAC resources to appear under, then type or paste the following URL: <code>https://podaac-tools.jpl.nasa.gov/drive/files</code></p>
    <p>Check <strong>Connect using different credentials</strong> and click <strong>Finish</strong></p>
    <p>When prompted to log in, use your URS username <strong></strong> and WebDAV password <strong></strong>.</p>
    <p>You can now browse PODAAC files and copy them to your computer just as you would with a local hard drive. It's best to copy a file to your computer before opening it.</p>
    <p>To disconnect, right-click the drive and select <strong>Disconnect</strong>.</p>
    <p>If you experience a slow connection or issues logging in, try going to <strong>Control Panel</strong>/<strong>Internet Options</strong>, select the <strong>Connections</strong> tab, click <strong>LAN settings</strong> and uncheck the <strong>Automatically detect settings</strong> box. For more information, see <a target="_blank" href="https://support.microsoft.com/en-us/kb/2445570">https://support.microsoft.com/en-us/kb/2445570</a>.</p>
    <p>For more information, see: <a target="_blank" href="http://windows.microsoft.com/en-us/windows/create-shortcut-map-network-drive#1TC=windows-7">Create a shortcut to (map) a network drive.</a></p>
    <hr>
    <h5>Connecting via Linux</h5>
    <p>Install <strong><a target="_blank" href="http://savannah.nongnu.org/projects/davfs2">davfs2</a></strong>. It's available in both the <strong>yum</strong> and <strong>apt-get</strong> package management systems.</p>
    <p>You can now use the <strong>mount</strong> command to mount PO.DAAC Drive as a drive of the type <strong>davfs</strong>. For example:</p>
    <p><code>mkdir /mnt/podaac_drive</code></p>
    <p><code>sudo mount.davfs https://podaac-tools.jpl.nasa.gov/drive/files /mnt/podaac_drive</code></p>
    <p>PO.DAAC Drive will now be available under the <code>/mnt/podaac_drive</code> path</p>
</div>
<hr>
<h5>Connecting via command line clients:</h5>
<p>"file_url" indicates the URL of the file in PO.DAAC Drive you want to download. You can get it by right-clicking on the file in the browser and selecting "Copy Link Address".</p>
<form class="form">
    <div class="form-group">
        CURL:
        <div>
            <textarea id="curl" class="form-control dav-snippet" type="text" readonly>curl -O -u username:password file_url</textarea>
        </div>
    </div>
    <div class="form-group">
        WGET:
        <div>
            <textarea id="wget" class="form-control dav-snippet" type="text" readonly>wget --user=username --password=password file_url</textarea>
        </div>
    </div>
    <div class="form-group">
        <p>Aria2:</p>
        Aria2 is a multi-threaded downloading tool that works similarly to WGET and CURL, but can be much faster with large files. Windows, Linux, and Mac OS X versions available at <a target="_blank" href="https://aria2.github.io/">https://aria2.github.io/</a>.
        <div>
            <textarea id="aria2" class="form-control dav-snippet" type="text" readonly>aria2c --http-user=username --http-passwd=password file_url</textarea>
        </div>
    </div>
</form>

						<h4>What is Earthdata Login?</h4>
						NASA's Earth Observation System Data and Information
						System (EOSDIS) Earthdata Login provides a simple,
						centralized mechanism for user registration and user
						profile management for all Distributed Active Archive
						Center (DAAC) users accessing data.
						<h4>Why do users have to register?</h4>
						The information provided by users when they register for
						Earthdata Login helps NASA better understand how our
						data are being used and the diverse needs of multiple
						user communities. This helps us prioritize improvements
						in data discovery, access, and usability. Registration
						also allows us to inform users with pertinent
						information such as the availability of new or updated
						data products, or data quality issues. In the future, it
						will allow for new value-added features and customized
						services resulting in a better user experience. For
						questions regarding Earthdata Login, see
						<a
							target="_blank"
							href="https://urs.earthdata.nasa.gov/documentation"
							>https://urs.earthdata.nasa.gov/documentation</a
						>.

	<script>
		window.addEventListener("load", function() {
			const menuNode = document.getElementById("contentsMenu");
			const helpSections = document.getElementById("helpSections");
			const helpHeader = document.getElementById("helpHeader");
			const helpHeaderText = document.getElementById("helpHeaderText");
			const helpReturnIcon = helpHeader.querySelector("i");
			function returnToMenu() {
				if (!menuNode.classList.contains("hidden")) return;
				helpHeaderText.innerText = "Help";
				helpReturnIcon.classList.add("hidden");
				helpSections.classList.add("hidden");
				menuNode.classList.remove("hidden");
			}
			function handleItemClick(e) {
				menuNode.classList.add("hidden");
				helpHeaderText.innerText = e.target.innerText;
				helpReturnIcon.classList.remove("hidden");
				helpSections.classList.remove("hidden");
				helpSections
					.querySelectorAll(".help-section")
					.forEach(node => node.classList.add("hidden"));
				document
					.getElementById(e.target.dataset.target)
					.classList.remove("hidden");
			}

			helpHeader.addEventListener("click", returnToMenu);
			document
				.querySelectorAll(".selectionButton")
				.forEach(button =>
					button.addEventListener("click", handleItemClick)
				);
		});
	</script>
